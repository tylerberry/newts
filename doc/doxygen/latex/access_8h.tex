\subsubsection{\setlength{\rightskip}{0pt plus 5cm}int access\_\-compare (const struct {\bf access} $\ast$ {\em one}, const struct {\bf access} $\ast$ {\em two})}\label{access_8h_cf3f0424886747ca4679dae6fbc45e8d}


Compare two \doxyref{access}{p.}{structaccess} rules, {\em one\/} and {\em two\/}, by the standard sorting rules used by the original notesfiles: first sorting by scope, where the ordering is user $<$ group $<$ system, then alphabetically by name. The name \char`\"{}other\char`\"{} is treated specially, and always sorts to the bottom. The name comparison is case-sensitive. Suitable as an argument to qsort(3) or other functions using the qsort interface.

\begin{Desc}
\item[Returns:]\begin{itemize}
\item -1 if {\em one\/} is \char`\"{}less than\char`\"{} than {\em two\/}. \item 0 if {\em one\/} and {\em two\/} are \char`\"{}equal\char`\"{}. \item 1 if {\em one\/} is \char`\"{}greater\char`\"{} than {\em two\/}. \end{itemize}
\end{Desc}
